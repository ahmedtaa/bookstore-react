# Book Store application

> The Bookstore is a website for listinf awesome books

- Display a list of books.
- Add a book.
- Remove a selected book.

## Built With

- HTML, CSS and JS
- ReactJS library

## Getting Started

### Install

` git clone git@github.com:ahmedtaa/bookstore-react.git YOUR_FOLDER_NAME`
` cd YOUR_FOLDER_NAME`
` npm start`
